# PSE协同办公系统
整体架构：前端主要通过jsp+jQuery+bootstrop。
后台采用SSM框架进行业务逻辑出路，后台数据库基于Mysql数据库进行开发。

## 项目流程
该项目为基于员工工作计划的审核查看系统。实现用户的登录注册功能的实现，用户等级权限的分配，人员等级基于zTree树的进行级别展示和分级控制、工作计划填报基于datatables可编辑表格以周为周期进行填报汇总。基于人员等级权限可查看本人及下属工作计划和工作日志。
